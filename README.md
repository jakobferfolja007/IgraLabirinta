# Pot tisočerih zavojev 🧩

Interaktivna spletna igra labirinta, kjer mora igralec najti pot skozi zapleten labirint v čim krajšem času. Igra omogoča več težavnostnih stopenj, merjenje časa in prikaz rešitve.

## 🎮 Funkcionalnosti

- Generiranje labirinta
- 4 stopnje težavnosti
- Merjenje časa reševanja
- Gumb za ponovni začetek (Reset)
- Prikaz rešitve labirinta
- Tabela rezultatov
- Enostaven in odziven uporabniški vmesnik
